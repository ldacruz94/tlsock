# Tlsock

I wanted to stop understanding TLS as a black box and actually try to implement it from scratch.
Nothing super in-depth, but basic enough to get a proper understanding of it all, so I'm refusing to use
frameworks doing the heavy lifting parts like connection handling or the handshake. 

It's small on purpose because this is a learning project and a portfolio piece second. No intention here on my end to
create a TLS library or anything that ambitious.
